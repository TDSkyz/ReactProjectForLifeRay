Folllow these step to run the application:
Step1: Go to the Back-end folder
Step2: Open terminal in this location
Step3: Type npm install
Step4:  Type npm start

Do the same thing with Front-End folder 

P/s: This application is using mongocloud DB, so you have to connect the internet to get the data. If you want to use local mongoDB, please change the config file in Back-end/server/config/database.config.js
